# Portfolio_Site

Curriculum-leadership portfolio site: an interactive standards-crosswalk auditor over my real day-by-day lesson plans. Live at **https://willbumgardner.com/curriculum-portfolio/**.

> Note: this directory is its own git repository (embedded in TheVault). Source of truth for the site lives here; the vault only tracks it as a gitlink.

## Pages

- [[README]] — what it does and why I built it.
- `index.html` — landing page.
- `crosswalk.html` (`crosswalk.js`) — per-course standards auditor: each indicator tagged *covered* / *gap* / *out of scope*, traced to the unit and day.
- `curriculum.html` (`curriculum.js`) — unit-by-unit course-map browser.
- `case-study.html` — curriculum-design case study.
- `style.css` — shared styles.

## Courses audited

Advanced CS I, Web Design & Development, Digital Game Development I & II, and CET — each mapped against its own grounded Nevada CTE standards catalog. Gap logic is scope-aware (out-of-level indicators show as out-of-scope, not gaps).
